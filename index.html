Projects

  • Modelo

    Modelo is cloud-based BIM platform which serves the best collaboration tool for architects. I designed and implemented its 3D rendering engine and cloud file translation server from scratch. Check out http://www.modelo.io for more information.

  • Bamboo

    Bamboo is an eBook rendering engine based on Paper3D and HTML5. It is written in C++ and cross-platform. With the command-line tool, users can easily compile a few HTML pages into a eBook archive which can be rendered by Bamboo reader. Check out Demo video.

  • Paper3D

    An open-source cross-platform (Android, iOS and Windows) rendering engine, best for creating the live wallpaper and casual TV games. Based on OpenGL ES 2.0 and C++, it provides a very easy-to-use interface that allows users to quickly develop an application from scratch. View project at GitHub and its GitPage.

  • Lens Flare Bundle

    Lens flare happens because of the interesting optics properties of the glasses when they interact the incoming lights. I implemented 10+ common lens flare effects in C++. Each of them is drawn with 2D lines, circles and filling. They can be combined together to form more interesting and fancy effects. More are here and GitHub project.
